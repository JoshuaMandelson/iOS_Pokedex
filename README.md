
iOS Pokédex App:

This is an iOS Pokédex built with SwiftUI, SwiftData, and MVVM, featuring Pokémon detail views and a persistent, user managed team of up to six Pokémon.

Features:

    •    Pokédex List
Displays Pokémon fetched from the PokéAPI using a SwiftUI list and navigation.

    •    Pokémon Detail View
Fetches and displays detailed Pokémon data including height, weight, and sprite images using async/await.

    •    Team Builder
Allows users to add Pokémon to a team with a maximum of six members.
Prevents duplicates and persists data using SwiftData.
Supports removing Pokémon from the team.

    •    User Feedback
Displays an alert when attempting to add a seventh Pokémon.
UI updates automatically across views.

    •    Tab-Based Navigation
Separate tabs for the Pokédex and Team views.
